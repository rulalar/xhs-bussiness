# 小红书业务集合网站

静态HTML网站，展示小红书业务相关内容。

## 项目结构
- `index.html` - 主页
- `styles.css` - 样式表
- `assets/` - 图片资源
- 各业务文件夹 - 包含各个业务线的内容

## 部署
此项目配置为在Vercel上部署。

## 本地测试
通过本地服务器可以测试网站功能。

## GitHub Desktop管理
使用GitHub Desktop可以方便地管理本项目：
1. 确保打开的是正确的仓库路径
2. 使用"Fetch origin"按钮获取最新更改
3. 提交更改后点击"Push origin"推送到远程仓库
